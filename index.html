QSA Linear and Quadratic Analysis in the Frequency Domain
Research Project
QSA is a mathematical method in neuroscience that aims to probe linear and quadratic neuronal functions in the frequency domain. It provides compact descriptions of the fundamental membrane properties for real neurons measured in experiments as well as for computational models. This project is being developed by Christophe Magnani and Lee E. Moore at the CNRS (The National Center for Scientific Research, France).

Documentation
QSA analysis is supplied for Python and RTXI. The Python instructions and reference describe how to compute QSA analysis with a little programming. The RTXI instructions describe the QSA protocol for doing RTXI experiments that can analyzed subsequently with Python.
